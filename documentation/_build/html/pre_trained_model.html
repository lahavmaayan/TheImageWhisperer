The Pre Trained Model¶


class TheNetwork.pre_trained_model.PreTrainedModel[source]¶

This class holds the Keras architecture of a neural network inspired by VGG. This network takes as input a 32 X 32 X 3 array and classifies to the 100 categories in CIFAR100.

This architecture was copied from https://github.com/geifmany/cifar-vgg

build_pre_trained_model()[source]¶

Build architecture and load weights of pre-trained network.

build_model()[source]¶

Build the network of vgg for 10 classes with massive dropout and weight decay as described in the paper.

load_weights()[source]¶

Load weights for pre-trained model.
