Welcome to TheImageWhisperer’s documentation!¶


TheImageWhisperer is a steganography detection app.

In short, steganography is a the practice of hiding a secret message inside a photo, such that the photo does not seem manipulated.

TheImageWhisperer detects such malicious photos!

This project was conceived and implemented at the DataHack2018 hackathon, and its source code is available here.
